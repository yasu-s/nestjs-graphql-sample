# 概要

- NestJS + GraphQL のサンプルです。

# 使用ライブラリ

- @nestjs/common - 7.4.x
- @nestjs/core - 7.4.x
- @nestjs/graphql - 7.6.x
- @nestjs/platform-express - 7.4.x
- apollo-server-express"- 2.16.x
- graphql - 15.3.x
- graphql-tools - 6.0.x

# 動作確認

## 1. サンプルのダウンロード

```bash
git clone git@github.com:yasu-s/nestjs-graphql-sample.git
```

## 2. パッケージインストール  

```bash
cd nestjs-graphql-sample
yarn install
```

## 3. サンプルの起動  

```bash
yarn start
```
